# Geometric Runner

A simple, endless runner platformer game built with HTML5 Canvas and Vanilla JavaScript.

## Features
- **Controls**: WASD / Arrow Keys / Space to Jump, Duck, and Move.
- **Gameplay**: Infinite scrolling world with obstacles.
- **Mechanics**:
    - **Jump** over standard obstacles.
    - **Duck** under floating obstacles.
    - **Bonuses**: Collect fruits for Score (+50), Speed Up, or Slow Down.

## How to Play
1. Open `index.html` in your browser.
2. Press any key to start.
3. Survive as long as possible!

## Deployment
This project is a static site (HTML/CSS/JS).

**Deploy to Netlify:**
1. Drag and drop the `Platform-game` folder into the Netlify Dashboard.
2. That's it!

**Deploy to GitHub Pages:**
1. Push to GitHub.
2. Enable GitHub Pages in Settings -> Pages -> Source: `main` / `root`.
